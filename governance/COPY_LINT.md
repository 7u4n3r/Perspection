# COPY LINT: SOVEREIGNTY & PERSPECTION

## Auto-fail if copy includes

- **Guidance language:** guide, help you, walk you through
- **Recovery language:** healing journey, progress, growth path
- **Ecosystem language:** aligned with, part of, integrated into
- **Obligation verbs:** should, need to, must, next step
- **Temporal pressure:** now, soon, don't miss

## Required posture

- Observational
- Optional
- Non-sequential
- Non-optimizing

## Single test

**If this sentence could make someone feel late, wrong, or behind, it fails.**
