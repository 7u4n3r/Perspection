# PUBLISH_CHECKLIST.md

## Purpose

Pre-publish checklist for Perspection public surfaces.

Use this before turning `public/INDEX_COPY.md` or any governance/reference material into a live page, reference homepage, documentation entry, or public governance surface.

## Current public copy source

`public/INDEX_COPY.md`

## Publish posture

Perspection may publish as a governance and reference surface for observation-only systems.

It may carry:

- System Refusal governance
- observation-only interface constraints
- copy lint rules
- audit workflows
- refusal constraints
- non-coercive orientation infrastructure
- implementation references

It should not become:

- Faceless Reclaimation
- Trinity Framework site
- Living Survival Guide
- ABLE-KY county map
- product funnel
- recovery sequence
- testimony archive
- public story body

## Pre-publish checklist

Before publishing, confirm:

- [ ] The page identifies itself as Perspection.
- [ ] The page describes governance/reference infrastructure only.
- [ ] The page does not claim to be Faceless Reclaimation.
- [ ] The page does not claim to be the Trinity Framework.
- [ ] The page does not present the entire ecosystem as one project.
- [ ] Related projects are labeled as related, not absorbed.
- [ ] The copy does not imply urgency.
- [ ] The copy does not imply the reader is late, wrong, behind, or incomplete.
- [ ] The copy does not instruct recovery.
- [ ] The copy does not create a progress path.
- [ ] The copy does not use obligation language.
- [ ] The copy passes `governance/COPY_LINT.md`.

## Voice check

The page may be:

- quiet
- observational
- reference-oriented
- non-sequential
- non-optimizing
- non-coercive
- structurally clear

The page should avoid becoming:

- motivational
- urgent
- therapeutic
- funnel-like
- story-heavy
- method-heavy
- over-explained
- ecosystem manifesto copy

## Boundary check

Allowed line:

> Perspection is governance and reference infrastructure for observation-only systems.

Allowed line:

> No action is required for the observation to remain valid.

Avoid lines like:

> Take back your story.
> Start here.
> First step.
> Practice this.
> Begin your journey.
> Powered by Trinity.
> Part of Faceless Reclaimation.
> You need to understand this.
